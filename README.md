# Spring Boot Todo Application

This is a simple todo application built using Spring Boot.

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Thymeleaf
- HTML
- CSS

## Features
- Add a new todo
- View all todos
- Update todo status
- Delete todo
